
To regenerate `../golocales.autogen.go`;

1. If needed, download the relevant CLDR version from [http://cldr.unicode.org/index/downloads](http://cldr.unicode.org/index/downloads). Pick the file named e.g. `cldr-common-48.1.zip`, and extract that file to the data directory.
2. Run either `go generate` from this folder of `go generate ./gen` from the root.
